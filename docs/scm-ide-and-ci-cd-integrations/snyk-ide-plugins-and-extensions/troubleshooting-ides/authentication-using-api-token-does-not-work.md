# API 토큰을 이용한 인증이 작동하지 않습니다

만약 **인증이 성공했다는 메시지를 본 뒤에 인증 오류가 발생한다면**, 인증 시 항상 동일한 주소에서 요청이 오도록 IPv4 또는 IPv6를 허용된 IP 주소 패밀리로 제한하는 것이 도움이 될 수 있습니다.
