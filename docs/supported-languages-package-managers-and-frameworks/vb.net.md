# VB.NET

## 적용 가능성

Snyk for VB.NET은 **Snyk Code에 대해서만 지원됩니다**.

Snyk 제품을 통해 가져올 수 있는 언어의 가용성을 확인하고 테스트하거나 모니터링할 수 있습니다.&#x20;

사용 가능한 기능:

* SCM 가져오기, 에서 가능함.&#x20;
* CLI 및 IDE를 통해 응용 프로그램을 테스트하거나 모니터링하고, 에서 가능함.&#x20;
