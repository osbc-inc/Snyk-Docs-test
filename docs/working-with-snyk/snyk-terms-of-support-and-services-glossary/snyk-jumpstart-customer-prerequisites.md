# Snyk Jumpstart 고객 사전 요구 사항

## Snyk Jumpstart 고객 사전 요구 사항 개요

SnykImplementation Consultant는 고객이 Snyk를 보다 빠르게 통합할 수 있도록 돕는 서비스를 제공하여, 지원 계정 구성을 통해 Snyk의 통합 가속화를 도울 것입니다. 본 참여는 지식 이전과 구성 과정을 통합합니다.

참여가 시작되기 전에, 각 제품 모듈에 대한 사전 요구 사항을 충족해야 합니다.

## Jumpstart에서 제외되는 항목

* Snyk Code 로컬 엔진
* Snyk Code 사용자 지정 규칙
* 드리프트 감지 구성
* API 스크립팅
* 20GB 이상의 모노 리포지토리
* 비공개 기능 (미공개 베타 버전 또는 초기 액세스 기능)
