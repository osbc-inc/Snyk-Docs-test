# 단계 2: Google 서비스 계정 (API) 생성

{% hint style="info" %}
**요약**\
Snyk를 위한 [Google 서비스 계정](https://cloud.google.com/iam/docs/service-accounts)을 선언하는 Terraform 템플릿을 다운로드했습니다. 이제 인프라를 프로비저닝하십시오.
