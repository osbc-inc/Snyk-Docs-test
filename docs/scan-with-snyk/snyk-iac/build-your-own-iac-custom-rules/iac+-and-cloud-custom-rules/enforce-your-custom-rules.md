# 사용자 정의 규칙 강제 적용하기

다음 CLI 명령어를 사용합니다:

```
snyk iac test
```

이 명령어를 사용하면 Snyk의 정책 엔진의 일부로 조직의 사용자 정의 규칙을 강제로 적용할 수 있습니다.
